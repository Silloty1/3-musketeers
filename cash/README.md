## Fork the project

1. To start this project, you need to fork it
2. After, you can clone the forked project in your repository ```https://github.com/YOUR_USERNAME/3-musketeers```

```sh
❯ cd /path/to/workspace
❯ git clone git@github.com:YOUR_USERNAME/3-musketeers.git
```

## Library installation
**For Windows**

Go on the cmd in your workspace, in /3-musketeers/cash and enter the following command :

```sh
npm install
```

## How to use the program
**For Windows**

Go on the cmd in your workspace, in /3-musketeers/cash and enter the following command :

```sh
node index.js 1 USD
```
You can enter any amount you want and the program will convert it for you. You can also select in which currency you want to convert by adding behind the ones your looking for, like the following command:

```sh
node index.js 1 USD EUR GBD CAD
```
